# 🚀 Portfolio Angel - Next.js Demo

Demo técnica creada para aprender **Next.js** y **Three.js**. Es un proyecto práctico donde desarrollo el portfolio de un compañero de DAM, enfocándome en rendimiento y animaciones fluidas.

## 🛠️ Tech Stack
* **Core:** Next.js (App Router) + TypeScript.
* **3D/FX:** Three.js (optimizado para mantener FPS estables).
* **UI:** Tailwind CSS + Shadcn/UI.
* **Deploy:** Configurado para GitHub Pages.

## ⚡ Setup
```bash
npm install
npm run dev
````
